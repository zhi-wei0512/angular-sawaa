「SAWAA，與肌膚的故事，始於潮汐。」

我們探尋到的第一個皮膚觀——是海岸——在「潮汐之地」——更了解保養的初衷：透過來自海洋與植萃的原料，讓肌膚沈浸在保養的過程當中，調理肌膚達到平衡，就如同自然永恆流動帶來的和諧，讓肌膚如潮汐變化一般溫和代謝。

「這就是保養的本質。」

海岸，猶如陸地的肌表，這片水與地的瑰麗交界，成為海陸的主體連接重疊而融合，不停被沖刷、雕縷、浸洗出新的視覺樣貌。除了海岸一詞，我們也發現了更完美地詮釋這永恆流動現象的:「潮汐之地」(Tidal Lands)。對人類來說，「潮汐之地」(Tidal Lands)是因著宇宙地、月之間萬有引力，不斷舞動且栩栩如生的見證。潮汐的漲落，搖晃著它銀色海潮的波面與光芒，過濾而包覆著海沙，如同肌膚紛紛圍繞鬆軟沙堆，浸潤形狀，消失，舒坦。
